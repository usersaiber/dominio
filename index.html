Dicas do Microsoft Excel

  • Use "Ctrl + N" para abrir uma nova planilha.
  • Fórmula SOMA: `=SOMA(A1:A10)` para somar células.
  • Conte células com `=CONTAR(A1:A10)`.

Dicas do Microsoft Word

  • Crie documentos novos com "Ctrl + N".
  • Negrito com "Ctrl + B".
  • Insira uma quebra de página com "Ctrl + Enter".
